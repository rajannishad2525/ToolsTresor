# ToolsTresor

A free online toolkit with **19+ privacy & utility tools** built with Flask (Python).

## Tools Available

| Tool | Description |
|---|---|
| Word Counter | Count words, characters, sentences |
| QR Generator | Generate QR codes instantly |
| Case Converter | UPPER, lower, Title, Sentence case |
| MD5 Generator | Generate MD5 hash of any text |
| Password Generator | Secure random password generator |
| Lorem Ipsum | Generate dummy placeholder text |
| What is My IP | Find your public IP address |
| Random Quote | Get a random inspirational quote |
| Header Checker | Inspect HTTP headers of any URL |
| Image Compressor | Compress images to reduce size |
| Steganography | Hide or extract secret messages in images |
| EXIF Cleaner | Remove metadata from images |
| URL Scanner | Unshorten and scan redirect URLs |
| Secret Note | Self-destructing one-time notes |
| Password Strength | Check how strong your password is |
| File Hasher | Get MD5 & SHA256 hash of any file |
| Digital Footprint | See what your browser reveals about you |
| PDF Cleaner | Remove metadata from PDF files |
| Image Watermarker | Add text watermark to images |
| File Locker | Encrypt and decrypt files with a key |

## How to Run

**1. Clone the repository**
```
git clone https://github.com/rajannishad2525/ToolsTresor.git
cd ToolsTresor
```

**2. Install dependencies**
```
pip install -r requirements.txt
```

**3. Run the app**
```
python app.py
```

**4. Open in browser**
```
http://localhost:5000
```
